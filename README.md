## 🚀 Key Features

- **Responsive Design**: Fluid and optimized layout tailored for mobile devices (360px–480px), tablets (e.g. iPad at 768px, 820px, 1024px), and ultra-wide desktops. Complete horizontal containment to prevent page shifts.
- **Light & Dark Mode**: Symmetrical, seamless theme toggling using CSS variables and local storage caching.
- **Dynamic Contact Form**: Complete client-side validation and direct form delivery integrated with **EmailJS**, featuring custom loading indicators and state disables during delivery.
- **Toast Notification System**: Modern, custom-styled slide-in notifications for actions (success, error, information) replacing intrusive browser `alert()` popups.
- **Copy-to-Clipboard**: Quick inline copy button for the email address with success toast confirmations.
- **Direct Recruiter Actions**: Clickable email client redirection (`mailto:`), mobile call options (`tel:`), and Google Maps location lookups (`Noida, India • Open to Relocation`) configured securely in new tabs.
- **Custom 404 Page**: Stylized `404.html` fallback matching the site design and typography variables.
- **SEO Optimized**: Standardized Open Graph and Twitter Card tags with correct metadata attributes and a custom favicon.

## 🛠️ Local Development

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/tusharagrahari214/Tushar-Portfolio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Tushar-Portfolio
   ```
3. Run a local web server (e.g., using Python or VS Code Live Server):
   - With Python 3:
     ```bash
     python -m http.server 8080
     ```
4. Access the site in your browser at `http://localhost:8080`.
